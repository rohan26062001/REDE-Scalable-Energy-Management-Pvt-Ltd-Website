Protocol Analyzer

Objective:

  • Protocols IEC61850, DNP3, IEC-60870-5-101 / 104, OPC and Modbus.
  • Online support interacting with your team for an immediate and satisfactory solution.
  • Specialized for the Energy Sector.

Main Feature:

  • It allows to export data on the tests carried out.
  • Simulation of multiple IEDs simultaneously.
  • Protocols are DNP3, IEC 60870-5-101 / 103/104 and Modbus.
  • Self-simulation of signals in slave devices.
  • Commands Allows you to configure commands in advance to perform tests more quickly.
  • It allows a quick diagnosis of communication networks.
  • Display data in a simple way, with a time stamp, name, description and quality.

Operating System: Windows XP / XP 64 bit / Windows 7/7 64 bit / Windows 8/8 64 bit / Server 2003 / Server 2008 / Server 2012 Recommended hardware: CORE I3 processor, 4GB RAM and 5 GB free hard drive Prerequisites: NET Framework 4.0, Windows Installer 4.5
